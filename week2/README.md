# Week 2 – End-to-End Machine Learning Pipeline

## Overview

This project demonstrates an end-to-end Machine Learning pipeline using the **Tesla Deliveries and Production (2015–2025)** dataset. The workflow covers data preprocessing, exploratory data analysis (EDA), feature engineering, regression modeling, hyperparameter tuning, and time series forecasting.

---

## Objectives

- Perform data preprocessing and cleaning
- Explore and visualize the dataset
- Engineer meaningful features
- Build regression models to predict average vehicle price
- Optimize model performance using GridSearchCV
- Forecast future deliveries using ARIMA time series modeling

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Statsmodels

---

## Workflow

1. Data Loading & Inspection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Data Preprocessing
6. Linear Regression
7. Random Forest Regression
8. Hyperparameter Tuning (GridSearchCV)
9. Time Series Forecasting (ARIMA)

---

## Dataset

**Tesla Deliveries and Production Dataset (2015–2025)**

The dataset contains information related to:

- Year and Month
- Region
- Tesla Model
- Estimated Deliveries
- Production Units
- Average Price (USD)
- Battery Capacity
- Driving Range
- CO₂ Saved
- Charging Stations

---

## Results

- Successfully implemented an end-to-end machine learning workflow.
- Compared Linear Regression and Random Forest models.
- Applied hyperparameter tuning using GridSearchCV.
- Built an ARIMA model to forecast future estimated deliveries.
